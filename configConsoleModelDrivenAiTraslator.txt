Complete setup to run this project perfectly on another PC

1. Install required software
- VS Code
- Git
- .NET SDK 10.0.x (recommended: 10.0.103, same as your machine)
- Optional: .NET SDK 9.0.x if you want parity with your current machine (you have 9.0.313 too)

2. Clone and build
- git clone <your-repo-url>
- cd PACXTools
- dotnet build AugustoCRV.Tools.ConsoleModelDrivenAiTraslator.slnx -nologo -v:minimal

3. Run the app
- AugustoCRV.Tools.ConsoleModelDrivenAiTraslator.exe --help
- Or interactive mode (no args): run the same exe without parameters

4. Configure connections (required before gen/push)
- ai-translator conn ai create --name my-ai --type AzureOpenAi --deployment-endpoint https://<resource>.openai.azure.com/openai/deployments/<deployment> --api-key <secret>
- ai-translator conn dataverse create --name my-dv --url https://org.crm.dynamics.com --username user@contoso.com --password <secret>
- ai-translator conn ai select
- ai-translator conn dataverse select

5. Config files to bring/check in the repo
- ConsoleModelDrivenAiTraslator.config.json
- ConsoleModelDrivenAiTraslator.config.json
- Populate-DefaultConsoleModelDrivenAiTraslatorConfig.ps1

6. Optional but useful tooling
- Xml2Doc CLI for API docs generation:
  dotnet tool install -g Xml2Doc.Cli --add-source https://api.nuget.org/v3/index.json --ignore-failed-sources

7. Validate everything
- dotnet --list-sdks
- dotnet --info
- dotnet build AugustoCRV.Tools.ConsoleModelDrivenAiTraslator.slnx
- ai-translator --help

Important migration note
- The app stores selected connections and credentials under %APPDATA%\AugustoCRV\Tools\ConsoleModelDrivenAiTraslator.
- You can migrate that folder, but it contains sensitive connection data. Safer approach is to recreate connections on the new PC with the commands above.
